# PaintPower Engine — Roadmap

This roadmap outlines the planned development of the PaintPower Engine and the larger Coco xPaint ecosystem.

---

## Editors

### Paint Editor
- Basic tools (brush, eraser, fill)
- Layers system
- Color picker improvements
- Export formats

### Script/Text Editor
- Syntax highlighting
- Line numbers
- Auto‑save
- Plugin support

### Sound Player
- Improved UI
- Waveform display
- But absolutely no editing sounds!
- Seriously no sound editor, no music editor, no editor that has to do with sound at all!
- But playing audio is fine.

### Animation Editor (WXA)
- WXA file loading
- Timeline UI
- Scene editing
- Export to WXA

### Video Editor
- Basic timeline
- Frame preview
- Import/export stubs

---

## Engine Core

- Project loading/saving
- File format support
- Plugin/extension system
- Cross‑editor communication
- Undo/redo system

---

## Online Features

- Project linking to user accounts
- Uploading projects to server
- Overwriting existing server projects
- Unlinking projects
- Authentication checks

---

## Build & Deployment

- Cross‑platform publishing scripts  
- Automated build pipeline (future)
- Release packaging

---

## Documentation

- WXA format specification  
- Upload workflow  
- Publishing instructions  
- Developer setup guide

## Long‑Term Goals

- Full multi‑editor creative suite  
- Custom extension marketplace  
- Real‑time collaboration  
- Cloud project syncing
